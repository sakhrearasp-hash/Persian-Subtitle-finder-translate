# 🎬 (Groq Edition) زیرنویس‌یاب و مترجم هوشمند فارسی پرسپولیس

یک وب‌اپلیکیشن فول‌استک هوشمند، محلی و فوق‌العاده حرفه‌ای برای جستجو، تحلیل، ترجمه و بومی‌سازی زیرنویس‌های فیلم، سریال و ویدیو به زبان فارسی روان ایرانی با حفظ دقیق زمان‌بندی (فرمت SRT) که **تنها و به طور اختصاصی توسط موتور هوش مصنوعی قدرتمند Groq** پشتیبانی می‌شود.

این سامانه به طور ویژه برای مترجمان، فیلم‌دوستان و تولیدکنندگان محتوا طراحی شده است تا فرآیند خسته‌کننده ترجمه زیرنویس را با دقت بالا و سرعت خیره‌کننده انجام دهند.

---

## 🏛 ویژگی‌های کلیدی جدید (Key Features)

- **🔍 جستجوگر آنلاین دیالوگ‌های اصلی**: با وارد کردن نام فیلم، هوش مصنوعی دیالوگ‌های واقعی و معتبر آن اثر را (تا ۳۵ خط متوالی و پیوسته برای شبیه‌سازی کامل یک سکانس فیلم) در قالب استاندارد SRT فراخوانی می‌کند.
- **📥 راه‌های ورود منعطف (SRT Import Suite)**:
  - بارگذاری مستقیم فایل با درگ اند دراپ (کشیدن و رها کردن فایل SRT).
  - پیست کردن دستی متون SRT در یک ادیتور یکپارچه.
- **🤖 موتور ترجمه هوشمند فارسی اختصاصی با Groq**:
  - بومی‌سازی عبارات، شوخی‌ها، اصطلاحات و تکیه‌کلام‌های فرنگی به زبان فارسی روزمره و طبیعی ایرانی (نه ترجمه کلمه به کلمه و مکانیکی).
  - استفاده از استراتژی دسته‌بندی ۳۵ خطی جدید برای درک کامل کنایه‌ها، سرگرمی‌ها و پیوستگی معنایی بین دیالوگ‌های متوالی بدون ایجاد محدودیت یا قطع اتصال.
- **⏱ حفظ دقیق زمان‌بندی (SRT Timing Protection)**: هیچ آسیبی به تگ‌های زمانی و فریم‌ها وارد نمی‌شود و زیرنویس نهایی کاملاً همگام باقی می‌ماند.
- **📥 دانلود دوطرفه فایل زیرنویس**:
  - امکان دانلود مستقیم فایل زیرنویس اصلی (Original SRT) پس از یافتن یا بارگذاری.
  - امکان دانلود فایل زیرنویس نهایی فارسی ترجمه‌شده و بومی‌سازی شده.
- **📝 ویرایشگر درون‌برنامه‌ای زنده**: هر خط ترجمه شده را به صورت تعاملی بازبینی و درجا اصلاح کنید.
- **🌌 پس‌زمینه سینمایی تخت جمشید در شب**: تم تاریک حرفه‌ای آراسته شده با نمایی هنری، تماشایی و با غلظت رنگ ملایم از ستون‌های شکوهمند تخت جمشید (پرسپولیس) در زیر آسمان پرستاره شب.

## 🤖 مدل‌های مورد پشتیبانی Groq

این سیستم به طور کامل و انحصاری با مدل‌های برتر و رسمی Groq هماهنگ شده است:
- `llama-3.3-70b-versatile` (قوی‌ترین و دقیق‌ترین مدل پیشنهادی)
- `llama-3.1-70b-versatile` (پایدار و باکیفیت بالا)
- `llama-3.1-8b-instant` (فوق‌العاده سریع و اقتصادی)
- `mixtral-8x7b` (مدل ترکیبی قدرتمند با درک زمینه خلاقانه)

## ⚙️ معماری و فناوری‌های به کار رفته (Tech Stack)

- **فرانت‌اند**: React 18، Vite، Tailwind CSS 4، Motion (برای انیمیشن‌های روان و چشم‌نواز)، آیکون‌های زیبای Lucide React.
- **بک‌اند**: Node.js با فریم‌ورک Express جهت کنترل ایمن کلیدهای API و ارتباط با پلتفرم هوش مصنوعی Groq.
- **هوش مصنوعی**: اتصال مستقیم و بدون واسطه به API رسمی Groq از طریق نقطه اتصال ایمن سرور.

## 🛠 راهنمای نصب و اجرای قدم‌به‌قدم محلی (Installation & Local Run Guide)

### پیش‌نیازها
مطمئن شوید که ابزار [Node.js](https://nodejs.org) (ترجیحاً نسخه ۱۸ یا بالاتر) بر روی سیستم عامل شما نصب شده باشد.

### مرحله ۱: کلون کردن مخزن یا استخراج فایل‌ها
پروژه را در پوشه دلخواه خود دریافت کنید:
```bash
git clone https://github.com/your-username/persian-subtitle-finder.git
cd persian-subtitle-finder
```

### مرحله ۲: نصب پکیج‌ها و وابستگی‌ها
دستور زیر را در خط فرمان یا ترمینال اجرا کنید تا کتابخانه‌ها به طور کامل نصب شوند:
```bash
npm install
```

### مرحله ۳: پیکربندی کلیدهای امنیتی (API Key Setup)
یک فایل تنظیمات با نام `.env` در پوشه اصلی پروژه بسازید و مقدار کلید اختصاصی Groq خود را در آن ذخیره کنید:
```env
GROQ_API_KEY="gsk_xxxxxxxxxxxxxxxxxxxxxxxx"
```
*نکته:* این برنامه مجهز به یک کادر اختصاصی در رابط کاربری است؛ به این معنی که کاربران می‌توانند کلیدهای شخصی خود را مستقیماً درون وب‌اپلیکیشن نیز وارد و ذخیره کنند.

### مرحله ۴: اجرای برنامه در محیط توسعه (Local Development)
برای راه‌اندازی سرور محلی و بارگذاری رابط کاربری به صورت زنده، دستور زیر را اجرا کنید:
```bash
npm run dev
```
سپس مرورگر خود را باز کرده و به آدرس زیر بروید:
```
http://localhost:3000
```

### مرحله ۵: تولید نسخه نهایی و اجرا در حالت پروداکشن (Build & Production Start)
برای ایجاد کدهای بهینه‌سازی شده بدون کدهای توسعه و اجرای سریع و روان روی لوکال‌هاست:
```bash
# بیلد کدهای فرانت‌اند و بک‌اند به صورت فشرده
npm run build

# اجرای سرور محصول
npm run start
```

## 💡 نکات و خطایابی (Troubleshooting)

- **خطای "کلید API یافت نشد"**: لطفاً مطمئن شوید که کلید معتبری را در پنل تنظیمات هوش مصنوعی برنامه وارد کرده‌اید یا فایل `.env` را به درستی ساخته‌اید.
- **ترجمه کل زیرنویس**: اکنون با بهینه‌سازی کدهای بخش ترجمه، دسته‌بندی‌ها به جای ۱۰ خطی به صورت ۳۵ خطی ارسال می‌شوند که این امر فرآیند ترجمه کامل فایل‌های سنگین را به شدت سریع‌تر کرده و از بروز خطاهای محدودیت یا گسستگی متن جلوگیری می‌کند.
- **دانلود فایل اصلی**: اگر مایلید نسخه اصلی انگلیسی/فرانسوی زیرنویس را پس از یافتن روی سیستم خود ذخیره کنید، اکنون دکمه زرد رنگ «دانلود زیرنویس اصلی» در بخش خروجی در دسترس شماست.
- **دکمه بازنشانی حافظه**: اگر احساس کردید که داده‌های قبلی مرورگر با کارهای جدید شما تداخل دارد، روی دکمه سرخ‌رنگ «بازنشانی حافظه موقت» در هدر کلیک کنید تا تمام کش پاک شود.
