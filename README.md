# 🎬 Persian Subtitle Finder (Ollama Edition) - زیرنویس‌یاب و مترجم هوشمند فارسی

یک وب‌اپلیکیشن فول‌استک هوشمند، ۱۰۰٪ محلی و فوق‌العاده حرفه‌ای برای جستجو، تحلیل، ترجمه و بومی‌سازی زیرنویس‌های فیلم، سریال و ویدیو به زبان فارسی روان ایرانی با حفظ دقیق زمان‌بندی (فرمت SRT) که **تنها توسط موتور هوش مصنوعی محلی Ollama** پشتیبانی می‌شود. این نرم‌افزار بدون نیاز به اینترنت (پس از دریافت مدل‌ها) و بدون نیاز به هیچ‌گونه کلید API کار می‌کند.

---

## 🏛 ویژگی‌های کلیدی (Key Features)

- **۱۰۰٪ آفلاین و امن**: بدون هیچ‌گونه وابستگی به سرویس‌های ابری (مانند Gemini یا ChatGPT). حریم خصوصی کامل.
- **🔍 جستجوگر آنلاین دیالوگ‌های اصلی**: پشتیبانی و شبیه‌سازی اختصاصی از ۴ موتور برتر جستجوی زیرنویس شامل:
  - OpenSubtitles
  - Subscene
  - YIFY Subtitles
  - Addic7ed
- **📥 راه‌های ورود منعطف (SRT Import Suite)**:
  - بارگذاری مستقیم فایل با درگ اند دراپ (کشیدن و رها کردن فایل SRT).
  - پیست کردن دستی متون SRT در یک ادیتور یکپارچه.
- **🤖 ترجمه هوشمند با Ollama**:
  - بومی‌سازی عبارات و اصطلاحات به زبان فارسی روزمره و طبیعی ایرانی (ترجمه مفهومی).
  - ترجمه کل فایل زیرنویس با استراتژی دسته‌بندی ۳۵ خطی برای درک کنایه‌ها و پیوستگی معنایی.
- **⏱ حفظ دقیق زمان‌بندی (SRT Timing Protection)**: هیچ آسیبی به تگ‌های زمانی وارد نمی‌شود.
- **📥 دانلود خروجی‌ها**: امکان دانلود زیرنویس اصلی و زیرنویس ترجمه‌شده.
- **📝 ویرایشگر زنده**: ویرایش و بازبینی خط به خط.
- **🌌 رابط کاربری سینمایی**: تم تاریک با پس‌زمینه تخت جمشید.

## 🤖 مدل‌های پیشنهادی Ollama

این سیستم با هر مدلی که در Ollama نصب داشته باشید کار می‌کند. پیشنهادهای ما:
- `llama3.1` (پیشنهادی - دقیق و عالی)
- `llama3.2` (مدل سبک و جدید)
- `mistral` / `mixtral` (سریع و قدرتمند)
- `qwen2` (درک بسیار خوب زبان‌های مختلف)
- `gemma` (مدل متن‌باز گوگل)

## 🛠 نصب و راه‌اندازی (فارسی)

### پیش‌نیازها
- نصب [Node.js](https://nodejs.org) (نسخه ۱۸ یا بالاتر)
- نصب [Ollama](https://ollama.com/) برای پردازش هوش مصنوعی محلی

### ۱. نصب وابستگی‌ها (Dependencies)
مخزن پروژه را کلون کرده یا دانلود کنید، سپس ترمینال را در پوشه پروژه باز کرده و دستور زیر را اجرا کنید:
```bash
npm install
```

### ۲. تنظیم Ollama Model
مدل هوش مصنوعی دلخواه خود را (مانند llama3.1) دریافت کنید:
```bash
ollama pull llama3.1
```
سپس آن را اجرا کنید تا سرور محلی Ollama روی پورت `11434` فعال شود:
```bash
ollama run llama3.1
```

### ۳. راه‌اندازی موتور جستجوی SearXNG (اختیاری برای وب اسکرپینگ واقعی)
در صورت نیاز به جستجوی زنده، می‌توانید یک نمونه محلی از SearXNG را با داکر اجرا کنید (فعلاً جستجو به صورت شبیه‌سازی‌شده در سرور قرار دارد).

### ۴. اجرای پروژه (Backend و Frontend)
با اجرای دستور زیر، هم سرور Node.js و هم رابط کاربری React همزمان راه‌اندازی می‌شوند:
```bash
npm run dev
```

### ۵. اجرا در مرورگر
مرورگر خود را باز کرده و به آدرس زیر بروید:
```
http://localhost:3000
```

## 🔄 به‌روزرسانی پروژه

**نسخه فعلی**: بهبود سیستم ترجمه مبتنی بر Ollama و اضافه شدن منابع جدید.

- **منابع جستجوی جدید**: تمرکز مستقیم بر روی پایگاه‌های معتبر شامل OpenSubtitles، Subscene، YIFY Subtitles و Addic7ed.
- **فیلترهای جستجوی هوشمندتر**: سیستم از ارسال نتایج غیرمرتبط جلوگیری کرده و صرفاً فایل‌های `.srt` یا صفحاتی که مستقیماً حاوی زیرنویس هستند را ارزیابی می‌کند.
- **ارتقای مدل Ollama**: پشتیبانی از مدل‌های بروزتر نظیر `llama3.2` و `qwen2.5` با قابلیت استدلال بالاتر برای حفظ لحن دیالوگ و روان بودن زبان فارسی اضافه شده است.

## 💡 نکات مهم

- **بدون اینترنت**: به محض اینکه مدل Ollama را دانلود کردید، می‌توانید ارتباط سیستم با اینترنت را قطع کنید و برنامه به کار خود ادامه می‌دهد.
- **تغییر مدل**: اگر مدل دیگری (مثل gemma) را نصب کرده‌اید، می‌توانید نام آن را در رابط کاربری انتخاب کنید تا برنامه به صورت خودکار از آن استفاده کند.
- **ترجمه دسته‌ای**: برنامه به گونه‌ای طراحی شده تا فایل کامل زیرنویس را پردازش کند. بسته به قدرت سخت‌افزار (کارت گرافیک/پردازنده)، ترجمه کامل یک فایل چند دقیقه‌ای طول خواهد کشید.
