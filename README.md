# 🎬 Persian Subtitle Finder & translate
 زیرنویس‌یاب و مترجم هوشمند فارسی

یک وب‌اپلیکیشن فول‌استک هوشمند، فوق‌العاده حرفه‌ای برای جستجو، تحلیل، ترجمه و بومی‌سازی زیرنویس‌های فیلم، سریال و ویدیو به زبان فارسی روان ایرانی با حفظ دقیق زمان‌بندی (فرمت SRT). این نرم‌افزار هم قابلیت کار به صورت ۱۰۰٪ محلی و آفلاین (از طریق Ollama) و هم قابلیت استفاده از سرویس‌های سریع ابری (مانند Cerebras) را دارد.

---

## 🏛 ویژگی‌های کلیدی (Key Features)

- **انعطاف در سرویس‌های هوش مصنوعی (AI Providers)**: 
  - **Ollama**: ۱۰۰٪ آفلاین و امن برای حفظ کامل حریم خصوصی.
  - **Cerebras**: مدل‌های سریع و ابری با تاخیر بسیار پایین.
- **🔍 جستجوگر آنلاین دیالوگ‌های اصلی**: پشتیبانی اختصاصی از موتورهای برتر جستجوی زیرنویس شامل:
  - OpenSubtitles (با امکان ورود کلید API شخصی)
  - موتور جستجوی عمومی وب (Free Web Engine بدون نیاز به API)
  - Subscene, YIFY Subtitles, Addic7ed
- **📥 راه‌های ورود منعطف (SRT Import Suite)**:
  - بارگذاری مستقیم فایل با درگ اند دراپ (کشیدن و رها کردن فایل SRT).
  - پیست کردن دستی متون SRT در یک ادیتور یکپارچه.
- **🤖 ترجمه هوشمند**:
  - بومی‌سازی عبارات و اصطلاحات به زبان فارسی روزمره و طبیعی ایرانی (ترجمه مفهومی).
  - ترجمه کل فایل زیرنویس با استراتژی دسته‌بندی خطوط برای درک کنایه‌ها و پیوستگی معنایی.
- **🛠 اصلاح خودکار (Auto-Correct)**:
  - تصحیح قواعد نگارشی و افزودن دقیق نیم‌فاصله‌های فارسی به زیرنویس‌ها.
- **⏱ حفظ دقیق زمان‌بندی (SRT Timing Protection)**: هیچ آسیبی به تگ‌های زمانی وارد نمی‌شود.
- **📝 ویرایشگر زنده**: ویرایش و بازبینی خط به خط.
- **🌌 رابط کاربری سینمایی**: تم تاریک با پس‌زمینه تخت جمشید.

## 🤖 مدل‌های پیشنهادی Ollama (در صورت استفاده محلی)

این سیستم با هر مدلی که در Ollama نصب داشته باشید کار می‌کند. پیشنهادهای ما:
- `llama3.1` (پیشنهادی - دقیق و عالی)
- `llama3.2` (مدل سبک و جدید)
- `mistral` / `mixtral` (سریع و قدرتمند)
- `qwen2` (درک بسیار خوب زبان‌های مختلف)

## 🛠 نصب و راه‌اندازی (فارسی)

### پیش‌نیازها
- نصب [Node.js](https://nodejs.org) (نسخه ۱۸ یا بالاتر)
- در صورت تمایل به استفاده کاملا آفلاین: نصب [Ollama](https://ollama.com/)

### راه‌اندازی سریع با یک کلیک (فقط برای ویندوز) ⚡
برای اجرای سریع پروژه در **ویندوز ۱۰ و ۱۱**، تنها کافی است فایل اجرایی زیر را که در پوشه پروژه قرار دارد اجرا کنید:
- دو بار کلیک روی فایل `start-windows.bat`
این فایل به صورت خودکار نصب پیش‌نیازها، بیلد پروژه، اجرای سرور و باز کردن مرورگر را برای شما انجام می‌دهد. نیازی به وارد کردن دستورات در ترمینال نیست!

### راه‌اندازی دستی (برای مک، لینوکس و ویندوز)

#### ۱. نصب وابستگی‌ها (Dependencies)
مخزن پروژه را کلون کرده یا دانلود کنید، سپس ترمینال را در پوشه پروژه باز کرده و دستور زیر را اجرا کنید:
```bash
npm install
```

#### ۲. تنظیمات هوش مصنوعی (انتخابی)
**اگر از Ollama استفاده می‌کنید:**
مدل هوش مصنوعی دلخواه خود را دریافت کنید:
```bash
ollama pull llama3.1
```
سپس آن را اجرا کنید:
```bash
ollama run llama3.1
```

**اگر از Cerebras استفاده می‌کنید:**
نیازی به نصب مدل ندارید، تنها در صفحه تنظیمات برنامه، کلید API خود را وارد کنید. مدلهای موجود به صورت خودکار از سایت Cerebras دریافت و نمایش داده می‌شوند.

#### ۳. اجرای پروژه (Backend و Frontend)
با اجرای دستور زیر، هم سرور Node.js و هم رابط کاربری React همزمان راه‌اندازی می‌شوند:
```bash
npm run dev
```

#### ۴. اجرا در مرورگر
مرورگر خود را باز کرده و به آدرس زیر بروید:
```
http://localhost:3000
```
