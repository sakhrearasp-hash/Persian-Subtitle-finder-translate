# 🎬 Persian Subtitle Finder (زیرنویس‌یاب و مترجم هوشمند فارسی)

یک وب‌اپلیکیشن فول‌استک هوشمند، محلی و فوق‌العاده حرفه‌ای برای جستجو، تحلیل، ترجمه و بومی‌سازی زیرنویس‌های فیلم، سریال و ویدیو به زبان فارسی روان ایرانی با حفظ دقیق زمان‌بندی (فرمت SRT) و سازگار با هوش مصنوعی.

این سامانه به طور ویژه برای مترجمان، فیلم‌دوستان و تولیدکنندگان محتوا طراحی شده است تا فرآیند خسته‌کننده ترجمه زیرنویس را با دقت بالا و سرعت خیره‌کننده انجام دهند.

---

## 🏛 ویژگی‌های کلیدی (Key Features)

- **🔍 جستجوگر آنلاین دیالوگ‌های اصلی**: با وارد کردن نام فیلم، هوش مصنوعی دیالوگ‌های واقعی و معتبر آن اثر را در قالب استاندارد SRT فراخوانی می‌کند.
- **📥 راه‌های ورود منعطف (SRT Import Suite)**:
  - بارگذاری مستقیم فایل با درگ اند دراپ (کشیدن و رها کردن فایل SRT).
  - پیست کردن دستی متون SRT در یک ادیتور یکپارچه.
- **🤖 موتور ترجمه هوشمند فارسی**:
  - بومی‌سازی عبارات، شوخی‌ها، اصطلاحات و تکیه‌کلام‌های فرنگی به زبان فارسی روزمره و طبیعی ایرانی (نه ترجمه کلمه به کلمه و مکانیکی).
  - ترجمه به صورت دسته‌ای (Batching) برای فهم کامل کنایه‌ها و پیوستگی معنایی بین دیالوگ‌های متوالی.
- **⏱ حفظ دقیق زمان‌بندی (SRT Timing Protection)**: هیچ آسیبی به تگ‌های زمانی و فریم‌ها وارد نمی‌شود و زیرنویس نهایی کاملاً همگام باقی می‌ماند.
- **📝 ویرایشگر درون‌برنامه‌ای زنده**: هر خط ترجمه شده را به صورت تعاملی بازبینی و درجا اصلاح کنید.
- **🧹 دکمه بازنشانی هوشمند حافظه**: پاکسازی تمام فایل‌ها و زیرنویس‌های موقت از روی مرورگر با فشردن یک دکمه، در حالی که کلید API شما برای استفاده‌های بعدی به صورت امن حفظ می‌گردد.
- **🌌 پس‌زمینه سینمایی تخت جمشید در شب**: تم تاریک حرفه‌ای آراسته شده با نمایی هنری، تماشایی و با غلظت رنگ ملایم از ستون‌های شکوهمند تخت جمشید (پرسپولیس) در زیر آسمان پرستاره شب.

---

## ⚙️ معماری و فناوری‌های به کار رفته (Tech Stack)

- **فرانت‌اند**: React 19، Vite، Tailwind CSS 4، Motion (برای انیمیشن‌های روان و چشم‌نواز)، آیکون‌های زیبای Lucide React.
- **بک‌اند**: Node.js با فریم‌ورک Express جهت کنترل ایمن کلیدهای API و ارتباط با مدل‌های هوش مصنوعی.
- **هوش مصنوعی**: کتابخانه رسمی SDK جمینای گوگل (`@google/genai`) با پشتیبانی اختیاری از مدل‌های OpenAI و Ollama (سرویس‌های محلی آفلاین).

## 🛠 راهنمای نصب و اجرای قدم‌به‌قدم برای گیت‌هاب (Installation & Local Run Guide)

### پیش‌نیازها
مطمئن شوید که ابزار [Node.js](https://nodejs.org) (ترجیحاً نسخه ۱۸ یا بالاتر) بر روی سیستم عامل شما نصب شده باشد.

### مرحله ۱: کلون کردن مخزن
ابتدا پروژه را در پوشه دلخواه خود دریافت کنید:
```bash
git clone https://github.com/your-username/persian-subtitle-finder.git
cd persian-subtitle-finder
```

### مرحله ۲: نصب پکیج‌ها و وابستگی‌ها
دستور زیر را در خط فرمان یا ترمینال اجرا کنید تا کتابخانه‌ها به طور کامل نصب شوند:
```bash
npm install
```

### مرحله ۳: پیکربندی کلیدهای امنیتی (API Key Setup)
یک فایل تنظیمات با نام `.env` در پوشه اصلی پروژه بسازید و مقدار کلید اختصاصی جمینای خود را در آن ذخیره کنید:
```env
GEMINI_API_KEY="کلید_اختصاصی_جمینای_شما"
```
*نکته:* این برنامه مجهز به یک کادر اختصاصی در رابط کاربری است؛ به این معنی که کاربران می‌توانند کلیدهای شخصی خود را مستقیماً درون وب‌اپلیکیشن نیز وارد کنند.

### مرحله ۴: اجرای برنامه در محیط توسعه (Local Development)
برای راه‌اندازی سرور محلی و بارگذاری رابط کاربری به صورت زنده، دستور زیر را اجرا کنید:
```bash
npm run dev
```
سپس مرورگر خود را باز کرده و به آدرس زیر بروید:
```
http://localhost:3000
```

### مرحله ۵: تولید نسخه نهایی و اجرا در حالت پروداکشن (Build & Production Start)
برای ایجاد کدهای بهینه‌سازی شده بدون کدهای توسعه و اجرای سریع و روان روی لوکال‌هاست:
```bash
# بیلد کدهای فرانت‌اند و بک‌اند به صورت فشرده
npm run build

# اجرای سرور محصول
npm run start
```

## 🔌 راهنمای اتصال به هوش مصنوعی آفلاین (Ollama Offline Integration)

اگر تمایل دارید ترجمه‌ها کاملاً رایگان، محلی و بدون نیاز به اینترنت روی سیستم شخصی شما انجام شود:
1. نرم‌افزار [Ollama](https://ollama.com) را دانلود و نصب کنید.
2. یکی از مدل‌های متناسب با ترجمه فارسی (مانند `llama3` یا `mistral` یا مدل‌های سبک دیگر) را با دستور زیر دریافت کنید:
   ```bash
   ollama run llama3
   ```
3. در وب‌اپلیکیشن «زیرنویس‌یاب هوشمند فارسی»، در منوی کشویی ارائه‌دهنده، گزینه **Ollama (آفلاین)** را برگزینید.
4. آدرس لوکال `http://localhost:11434` به صورت خودکار اعمال شده و ترجمه آغاز می‌گردد!

## 📚 ساختار و کدهای کلیدی پروژه (Project Structure)

- **`server.ts`**: سرور اصلی Express که درخواست‌ها را دریافت کرده، کدهای SRT را دسته بندی می‌کند و به سرورهای جمینای ارسال می‌نماید تا امنیت کلیدها کاملاً تضمین شود.
- **`src/App.tsx`**: کنترلر اصلی فرانت‌بند که دارای تم تاریک عریض، لایه شیشه‌ای (Glassmorphism)، مدیریت پویای وضعیت‌ها (States) و کادرهای تعاملی ویرایش است.
- **`src/utils/srtParser.ts`**: ماژول مهندسی معکوس و تحلیل فرمت فایل‌های SRT برای جداسازی نشانگرهای زمان و حفظ قالب اصلی.
- **`src/index.css`**: فایل استایل‌های پایه، ایمپورت فونت وب زیبای **وزیرمتن (Vazirmatn)** و پیکربندی متغیرهای رنگی اختصاصی Tailwind CSS 4.

## 💡 نکات و خطایابی (Troubleshooting)

- **خطای "کلید API یافت نشد"**: لطفاً مطمئن شوید که کلید معتبری را در پنل تنظیمات هوش مصنوعی برنامه وارد کرده‌اید یا فایل `.env` را به درستی ساخته‌اید.
- **قطع اتصال سرور یا فریز شدن در فایل‌های سنگین**: در صورت بارگذاری فصل‌های کامل سریال یا فیلم‌های طولانی، به دلیل حجم بالا پیشنهاد می‌شود از گزینه دسته بندی استفاده کنید که سرور Express ما به طور پیش‌فرض آن را به دسته‌های ۱۰ تایی تقسیم می‌کند تا از خطای محدودیت کاراکتر جلوگیری کند.
- **دکمه بازنشانی حافظه**: اگر احساس کردید که داده‌های قبلی مرورگر با کارهای جدید شما تداخل دارد، روی دکمه سرخ‌رنگ «بازنشانی حافظه موقت» در هدر کلیک کنید تا تمام کش پاک شود بدون اینکه نیاز به تایپ مجدد کلید API خود داشته باشید.
